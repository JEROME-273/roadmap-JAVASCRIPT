# Week 3: Control Flow & Decision Making

## Overview

Now that you can work with variables and functions, it's time to make your programs smart! This week you'll learn how to make decisions in code and repeat actions efficiently.

## What You'll Learn

- Making decisions with if/else statements
- Using comparison and logical operators
- Switch statements for multiple conditions
- For loops and while loops
- Breaking down complex logic
