# Week 1: Day 2 - Functions

## Learning Objectives:

- Understand what functions are and why they're useful
- Learn to write and call functions
- Practice with parameters and return values

### Theory:

Functions are reusable blocks of code that perform specific tasks. Think of them as mini-programs within your program.

```javascript
// Function declaration
function functionName(parameter) {
  // code to execute
  return result; // optional
}

// Function call
functionName(argument);
```
