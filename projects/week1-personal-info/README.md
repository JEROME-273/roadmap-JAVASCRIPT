# Your First JavaScript Project: Personal Information Display

## Project Overview

Build a simple program that collects and displays personal information. This project combines everything you've learned in Week 1.

## Requirements

Create a program that:

1. Stores personal information (name, age, hobbies, etc.)
2. Uses functions to organize code
3. Displays information in a formatted way
4. Performs some calculations

## Project Structure

```
personal-info/
├── index.html (for web version)
├── script.js (your JavaScript code)
└── style.css (basic styling)
```

## Step-by-Step Guide

### Step 1: Plan Your Program

Before coding, think about:

- What information do you want to display?
- What functions will you need?
- How will you organize your code?

### Step 2: Write the JavaScript

Complete the tasks in `script.js`

### Step 3: Create the HTML

Create a simple webpage to run your JavaScript

### Step 4: Test and Improve

Run your program and make improvements

## Success Criteria

✅ Program runs without errors
✅ Uses variables appropriately  
✅ Uses functions to organize code
✅ Displays information clearly
✅ Performs at least one calculation

## Bonus Challenges

- Add input validation
- Calculate and display your birth year
- Add more personal statistics
- Make the output more visually appealing

Remember: This is YOUR first project. Make it personal and have fun with it!
