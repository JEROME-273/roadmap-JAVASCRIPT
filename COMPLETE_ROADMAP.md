## ðŸ’¡ Study Tips

### Daily Routine (45-60 minutes)

1. **Read lesson** (10 mins) - Understand concepts
2. **Code along** (20 mins) - Type the examples
3. **Practice exercises** (20 mins) - Do the TODOs
4. **Review** (10 mins) - What did you learn?

### Weekly Routine

- **Monday-Friday**: Daily lessons (45 mins each)
- **Saturday**: Start weekend project
- **Sunday**: Finish project + review week

### When You're Stuck

1. Read error messages carefully
2. Check for typos
3. Use console.log() to debug
4. Google the error message
5. Take a break and come back
6. Ask for help (it's normal!)
